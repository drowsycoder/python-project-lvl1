##
This repository represents a simple game set called Brain Games as an educational project on Hexlet.
The set includes 5 modules:
1. Calc game: calculate the result of a simple arithmetic expression.
2. Even game: determine is a number even or not.
3. Gcd game: find the greatest common divisor of two numbers.
4. Prime game: determine is a number prime or not.
5. Progression game: find the missing element in an arithmetic progression.
